## 🚀 Cara Bermain

1. Klik **"Start New Game"**
2. Klik kartu untuk membuka
3. Cocokkan 2 kartu dengan emoji sama
4. Selesaikan semua 8 pasang
5. Setiap langkah mendapat **Proof Billions TEE**
